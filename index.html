New York Times Crossword Solver

Uses an api repository to supply a randomly selected New York Times crossword puzzle from the archive. The interface allows you to solve the puzzle with cards displaying the clues, and includes an answer checker to tell you if you have completed it successfully (and how many correct spaces you have if it is not finished). The archive of puzzles covers the years 1976-2018.

Catchphrase

An implementation of the board game catchphrase. When the game is started the timer ticks down, slowly increasing in speed, the objective is to get your team to guess the word on the device and then pass the device to the other team to do the same. Whoever is left holding the device when the timer runs out loses.

Tour Guide Robot

Senior robotics project. Implemented a system combining ROS with Amazon Alexa integration to allow a robot to give guided tours of an engineering facility using voice commands. The building was pre-mapped using the robot's sensors to create a proper 3d navigation map and the robot could then use location data to determine where it was in the facility and guide users to points of interest they asked about before reading prepared speeches on those locations.
